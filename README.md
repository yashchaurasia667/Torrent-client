# Torrent-client

## Installation

```bash
$ git clone https://github.com/yashchaurasia667/torrent-client
$ cd torrent-client
```

## Parser

```bash
$ make parser
$ ./bin/parser {name_of_torrent_file_to_decode}
```

## Test

```bash
$ make
```

## Download torrent

```bash
$ go run {path_to_torrent_file} {path_to_output_directory}
```
